# Acceptance Checklist

## Visual parity
✅ Layout matches original on desktop
✅ Layout matches original on mobile
✅ Fonts match (family, weight, sizes)
✅ Spacing matches (major sections)
✅ Colors match

## Content parity
✅ All sections present
✅ Text matches source (no rewrites)
✅ Images/icons match and load

## Language
✅ /en, /de, /ru exist
✅ Language switcher works globally
✅ Switching language keeps same page context

## Consent banner
✅ First visit shows banner
✅ Accept stores consent + prevents re-show
✅ Reject stores choice + prevents re-show
✅ Clearing storage restores banner
✅ Non-essential scripts blocked until accept
✅ Text localized EN/DE/RU

## Legal pages
✅ Impressum exists as separate page
✅ Privacy Policy exists as separate page
✅ Both linked in footer
✅ Both language-aware

## Build & runtime
✅ npm run build ok
✅ No console errors
✅ No broken routes
