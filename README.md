# Disterheft Site Mirror

## Install

```bash
npm install
```

## Dev

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Project Structure

- `src/layouts/`: shared page layouts
- `src/components/`: reusable UI components
- `src/content/`: content and copy files
- `src/styles/`: global and component styles
- `src/scripts/`: client-side scripts
- `src/pages/`: file-based routes for /en, /de, /ru
- `public/`: static assets
